\section{Analisis Kinerja Sistem} %//?
Mengingat sistem ini akan digunakan oleh banyak agen secara bersamaan, performa respons server menjadi parameter penting. Berdasarkan pengujian beban ringan menggunakan \textit{Postman}, rata-rata waktu respons untuk penyimpanan data penyewa adalah 120ms, dan pengambilan data historis (history log) berkisar antara 200ms--350ms tergantung jumlah data. Arsitektur Node.js yang bersifat \textit{asynchronous} terbukti efektif menangani permintaan I/O (\textit{Input/Output}) ke database MySQL tanpa memblokir proses lain, sehingga sistem tetap responsif meskipun beberapa agen melakukan input data secara bersamaan.
