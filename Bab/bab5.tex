\chapter{Implementasi Perangkat Lunak}
\label{chap:implemtasiPL}

Bab ini membahas implementasi dari aplikasi {Data Capture} berbasis Node.js sebagai bentuk perwujudan dari rancangan sistem sebelumnya. Penjelasan pada bab ini mencakup realisasi tampilan antarmuka perangkat lunak serta simulasi operasional sistem ketika memproses input data secara langsung. Simulasi tersebut secara spesifik mendemonstrasikan bagaimana sistem merespons dan membedakan antara input data penyewa yang berstatus aman dengan penyewa yang berpotensi melanggar.
